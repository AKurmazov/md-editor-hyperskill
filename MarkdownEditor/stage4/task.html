Description

Lists are really useful! For example, imagine yourself going to a grocery shop, I bet you would probably write some kind of a list of products you need to remember to buy. In markdown lists are as simple as they are in the example; actually, there only two different kinds: ordered and unordered. I think you've already guessed that the difference between them is that an ordered list enumerates its elements (1., 2., 3., and so on) and an unordered doesn't.

Remember those ordered and unordered list formatters we skipped in the last stage? Well, let's get back to them now!

Objectives

The goal is to implement the ordered and unordered list formatters. You may separate the implementation into two different functions for each of the formatters, but I suggest keeping them in one: try to get an idea on how to do it!

For both of them the following inputs are required:

	- Number of rows: > 3
- Row #1: > First element
- Row #2: > Second element
- Row #3: > Third element

Don't forget to check if the number of rows is less or equal to zero! If it is invalid print the corresponding error: The number of rows should be greater than zero

Examples

A working session with the tool might look like this after completing this stage. Notice that the formatters automatically add a new-line

	$ python markdown-editor.py
Choose a formatter: > ordered-list
- Number of rows: > 3
- Row #1: > First element
- Row #2: > Second element
- Row #3: > Third element

2. Second element
3. Third element

Choose a formatter: > unordered-list
- Number of rows: > 2
- Row #1: > Fourth element
- Row #2: > Fifth element

2. Second element
